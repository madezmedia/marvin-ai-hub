# Marvin's AI Learning Hub 🎤🤖

An interactive learning platform designed for Marvin (karaoke DJ, entertainer, and singer) to master AI tools and build an AI-powered entertainment brand.

## Features

- **7-Day Learning Roadmap** - Structured skill-building challenge with progress tracking
- **Official Resources** - Curated links to Claude, Gemini, and ChatGPT documentation
- **AI Tools Comparison** - Side-by-side comparison of top AI platforms
- **Collaboration Ideas** - AI + Entertainment workflow concepts
- **Interactive UI** - Beautiful, responsive design with smooth transitions

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **React Hooks** - State management

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to see the site.
